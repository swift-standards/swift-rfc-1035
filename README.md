# swift-rfc-1035

Swift implementation of [RFC 1035: Domain Names Implementation and Specification](https://www.rfc-editor.org/rfc/rfc1035.html)

## Installation

### Swift Package Manager

Add this to your Package.swift:

```swift
dependencies: [
    .package(url: "https://github.com/swift-web-standards/swift-rfc-1035.git", branch: "main")
]
```
